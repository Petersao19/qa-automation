Este proyecto permitió validar tanflujos E2E de interfaz como pruebas de API utilizando Cypress, 
asegurando el correcto funcionamiento de los principales escenarios de la aplicación.
La automatización del login mediante comandos personalizados facilitó la reutilización de código 
y mejoró la mantenibilidad de los tests.
Por otro lado, las pruebas de API permitieron validar respuestas, estructura y datos de endpoints 
REST de forma independiente a la interfaz.
En conjunto, el proyecto demuestra una correcta organización de pruebas, buenas prácticas 
de automatización y un enfoque orientado a escenarios reales de testing
